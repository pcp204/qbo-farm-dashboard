# QBO Farm Dashboard Public Paperwork Site

This repository is a static public GitHub Pages site for Intuit production key approval paperwork only.

It does not contain the private local Flask app. It does not contain `.env` files, `tokens.json`, API keys, QuickBooks data, CSV exports, farm financial data, or secrets.

QBO Farm Dashboard is a private internal farm management dashboard for Pustejovsky & Sons. It connects to QuickBooks Online only with authorized user consent. It is used for internal farm reporting, budgeting, liability payment review, and break-even analysis.

## Pages

- `index.html` - Site home
- `privacy.html` - Privacy policy
- `eula.html` - End User License Agreement
- `connect.html` - Local app connection instructions
- `disconnect.html` - Disconnect instructions
