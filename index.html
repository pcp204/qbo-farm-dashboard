QBO Farm Dashboard

QBO Farm Dashboard is a private internal farm management dashboard for Pustejovsky & Sons. It connects to QuickBooks Online only with authorized user consent. It is used for internal farm reporting, budgeting, liability payment review, and break-even analysis.

This public website provides paperwork pages required for Intuit production key approval. It does not contain the private local application, API keys, tokens, QuickBooks data, CSV exports, or farm financial data.
